# 🖼️ Duplicate Image Finder

A fast and lightweight desktop application for finding and managing duplicate images using perceptual image hashing.

Created by **Deepanshu Singh**

</div>

---

## ✨ Features

- 🔍 Scan folders recursively
- 🖼️ Detect duplicate images
- ⚡ Fast perceptual hashing
- 👀 Image preview
- 🗑️ Move duplicates to Trash
- 🎯 Easy-to-use interface
- 🖥️ Cross-platform support
- 📂 Supports large image collections
- 🚀 Standalone executable support
